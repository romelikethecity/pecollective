AI Token Pricing 2026 - Cost Per 1M Tokens for Every Major LLM

By Rome Thorndike · April 7, 2026 · 8 min read

Every LLM API charges by tokens. Understanding token pricing is the foundation of managing AI costs. This guide covers what tokens are, what they cost across every major provider, and how to estimate your monthly spend.

What Is a Token?

A token is the unit LLMs use to process text. One token is roughly 4 characters or 0.75 words in English. A page of text (500 words) is about 670 tokens. A full-length book (80,000 words) is about 107,000 tokens.

Key conversions:

  • 1 token = ~4 characters in English
  • 1,000 tokens = ~750 words
  • 1M tokens = ~750,000 words = ~7.5 average books
  • 100 words = ~133 tokens

Per 1K Tokens vs Per 1M Tokens

Older pricing pages quote costs per 1K tokens. Current standard is per 1M tokens. To convert: multiply per-1K price by 1,000 to get per-1M price. Example: GPT-4o at $0.0025/1K input = $2.50/1M input.

How to Reduce Token Costs

1. Use prompt caching

Anthropic, OpenAI, and Google offer prompt caching that reduces costs for repeated context by up to 90%. If your system prompt is 5,000 tokens and you send it with every request, caching eliminates that cost after the first call.

2. Use batch processing

OpenAI offers 50% discounts on batch API calls (non-real-time). If your workload can tolerate 24-hour latency, this cuts costs in half immediately.

3. Match model to task

Use the cheapest model that produces acceptable output. Route simple tasks (classification, extraction) to Nano/Flash and complex tasks (reasoning, creative writing) to full-size models. Many teams use a router that selects the model based on task complexity.

4. Limit output tokens

Set max_tokens in your API call. If you need a yes/no answer, don't let the model generate 500 tokens of explanation. Output tokens cost 3-5x more than input tokens.

5. Compress inputs

Remove unnecessary whitespace, redundant instructions, and boilerplate from prompts. A 20% reduction in prompt size directly reduces input token costs by 20%.

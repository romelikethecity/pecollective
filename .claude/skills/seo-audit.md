# SEO Audit Skill — PE Collective

Run this audit to check the SEO health of pecollective.com. Covers technical SEO, on-page SEO, structured data, and content quality.

## Domain & Hosting

- **Canonical Domain:** pecollective.com
- **Hosting:** GitHub Pages (deployed via Actions workflow)
- **DNS:** Cloudflare
- **SSL:** Enabled
- **Source of truth:** `site/` directory (deployed as GitHub Pages artifact)
- **Root `index.html`** is copied into `site/` by CI — both must stay in sync

## Tracking

PE Collective does not currently have GA4 or Clarity tracking. Flag this as a finding if still missing.

## SEO Checklist for Every Page

- [ ] Unique `<title>` tag (under 60 chars), includes primary keyword
- [ ] Unique `<meta name="description">` (120-160 chars)
- [ ] `<link rel="canonical">` set to `https://pecollective.com/{path}`
- [ ] Open Graph tags: og:title, og:description, og:type, og:url, og:image
- [ ] Twitter Card tags: twitter:card, twitter:title, twitter:description
- [ ] `<h1>` tag present and unique per page
- [ ] Page listed in `site/sitemap.xml`
- [ ] At least 2 internal links to other pages on the site
- [ ] No broken internal links
- [ ] Images have `alt` attributes
- [ ] Content wrapped in `<main>` landmark

## Audit Procedure

### 1. Technical SEO

```bash
# Check robots.txt exists and is correct
cat site/robots.txt

# Check sitemap exists and list URLs
cat site/sitemap.xml | grep "<loc>" | wc -l

# Check for pages NOT in sitemap
find site -name "index.html" -type f | sed 's|site/||' | sed 's|/index.html||' | sort > /tmp/pages.txt
grep "<loc>" site/sitemap.xml | sed 's|.*<loc>https://pecollective.com/||' | sed 's|</loc>||' | sed 's|/$||' | sort > /tmp/sitemap.txt
diff /tmp/pages.txt /tmp/sitemap.txt
```

### 2. Meta Tags Audit

```bash
# Check all pages for meta title presence and length
for f in $(find site -name "index.html" -type f); do
  title=$(grep -o '<title>[^<]*</title>' "$f" | head -1 | sed 's/<[^>]*>//g')
  len=${#title}
  if [ -z "$title" ]; then
    echo "MISSING TITLE: $f"
  elif [ $len -gt 60 ]; then
    echo "LONG TITLE ($len): $f — $title"
  fi
done

# Check meta descriptions
for f in $(find site -name "index.html" -type f); do
  desc=$(grep -o 'meta name="description" content="[^"]*"' "$f" | head -1 | sed 's/.*content="\(.*\)"/\1/')
  len=${#desc}
  if [ -z "$desc" ]; then
    echo "MISSING DESC: $f"
  elif [ $len -gt 160 ]; then
    echo "LONG DESC ($len): $f"
  elif [ $len -lt 120 ]; then
    echo "SHORT DESC ($len): $f"
  fi
done

# Check canonical URLs
for f in $(find site -name "index.html" -type f); do
  canon=$(grep -o 'rel="canonical" href="[^"]*"' "$f" | head -1)
  if [ -z "$canon" ]; then
    echo "MISSING CANONICAL: $f"
  fi
done
```

### 3. Open Graph & Twitter Cards

```bash
# Check OG tags
for f in $(find site -name "index.html" -type f); do
  og_title=$(grep -c 'og:title' "$f")
  og_desc=$(grep -c 'og:description' "$f")
  og_image=$(grep -c 'og:image' "$f")
  if [ "$og_title" -eq 0 ] || [ "$og_desc" -eq 0 ]; then
    echo "MISSING OG TAGS: $f (title:$og_title desc:$og_desc image:$og_image)"
  fi
done

# Check Twitter cards
for f in $(find site -name "index.html" -type f); do
  tw=$(grep -c 'twitter:card' "$f")
  if [ "$tw" -eq 0 ]; then
    echo "MISSING TWITTER CARD: $f"
  fi
done
```

### 4. Structured Data

```bash
# Check for JSON-LD on all pages
for f in $(find site -name "index.html" -type f); do
  ld=$(grep -c 'application/ld+json' "$f")
  if [ "$ld" -eq 0 ]; then
    echo "NO SCHEMA: $f"
  fi
done

# Validate JSON-LD is parseable
for f in $(find site -name "index.html" -type f); do
  python3 -c "
import re, json, sys
with open('$f') as fh:
    content = fh.read()
blocks = re.findall(r'<script type=\"application/ld\+json\">(.*?)</script>', content, re.DOTALL)
for i, block in enumerate(blocks):
    try:
        json.loads(block)
    except json.JSONDecodeError as e:
        print(f'INVALID JSON-LD in $f block {i}: {e}')
" 2>/dev/null
done
```

### 5. Heading Structure

```bash
# Check H1 tags
for f in $(find site -name "index.html" -type f); do
  h1_count=$(grep -c '<h1' "$f")
  if [ "$h1_count" -eq 0 ]; then
    echo "MISSING H1: $f"
  elif [ "$h1_count" -gt 1 ]; then
    echo "MULTIPLE H1s ($h1_count): $f"
  fi
done
```

### 6. Accessibility & Performance

```bash
# Check for images without alt text
for f in $(find site -name "index.html" -type f); do
  missing_alt=$(grep -c '<img[^>]*[^a]>' "$f" 2>/dev/null || echo 0)
  # More precise check
  no_alt=$(grep -oP '<img(?![^>]*alt=)[^>]*>' "$f" | wc -l)
  if [ "$no_alt" -gt 0 ]; then
    echo "IMG WITHOUT ALT ($no_alt): $f"
  fi
done

# Check for main landmark
for f in $(find site -name "index.html" -type f); do
  main=$(grep -c '<main' "$f")
  if [ "$main" -eq 0 ]; then
    echo "MISSING <main>: $f"
  fi
done
```

### 7. Internal Linking

```bash
# Count internal links per page
for f in $(find site -name "index.html" -type f); do
  links=$(grep -oP 'href="(?!https?://|#|mailto:)[^"]*"' "$f" | wc -l)
  if [ "$links" -lt 2 ]; then
    echo "LOW INTERNAL LINKS ($links): $f"
  fi
done
```

### 8. Tracking & Analytics

```bash
# Check for GA4
grep -rL "googletagmanager\|gtag" site/ --include="*.html" | head -20

# Check for any analytics
grep -rL "analytics\|clarity\|gtag" site/ --include="*.html" | head -20
```

### 9. Content Quality

- Read each page and check for AI-flagged writing patterns:
  - Em-dashes (heavily flagged)
  - "Unlock", "Unleash", "Empower", "Supercharge", "Revolutionize"
  - Overly parallel list structures
  - Rhetorical questions with immediate answers
  - "cutting-edge", "revolutionary", "leverage"

### 10. Cross-Page Consistency

- Nav links match `scripts/nav_config.py`
- Footer content is consistent across pages
- Root `index.html` matches `site/index.html`
